# 无人机虚拟仿真巡防管控系统 — 视频演示流程大纲

> 建议总时长：8-12 分钟（根据答辩要求调整）

---

## 一、开场（30秒）

**操作**：打开浏览器，进入系统登录页

**讲解**：
- "这是基于 SpringBoot + Vue 开发的无人机虚拟仿真巡防管控系统"
- "系统实现无人机设备管理、航线规划、任务模拟和结果归档的完整业务闭环"
- "现在开始演示"

---

## 二、登录与首页（1分钟）

**操作**：
1. 输入 admin 账号密码登录
2. 展示首页 Dashboard

**讲解**：
- "系统采用 JWT 令牌认证，基于若依框架的 RBAC 权限模型"
- "首页展示四大核心数据：设备总数、航线总数、任务总数、结果总数"
- "设备状态分布饼图直观展示正常/维修/报废/任务中的占比"
- "近7天任务趋势折线图反映巡防活动变化"
- "点击任意统计卡片可快速跳转到对应管理页面"

---

## 三、设备管理（1分钟）

**操作**：
1. 进入设备管理页
2. 展示设备列表（编号、名称、状态）
3. 点击"新增"录入一台设备
4. 点击某设备的"详情"，展示关联的历史任务

**讲解**：
- "设备管理支持增删改查，设备编号唯一校验"
- "设备状态包含正常、维修中、已报废、任务中、离线五种"
- "设备详情可查看该设备参与过的所有历史任务记录"
- "已绑定未完成任务的设备受保护，无法直接删除"

---

## 四、航线规划（1.5分钟）

**操作**：
1. 进入航线管理页
2. 点击"新增"，打开地图绘制弹窗
3. 在地图上搜索地点并定位
4. 点击"开始绘制"，在地图上点击多个点位形成航线
5. 填写航线名称、预计时长、飞行高度
6. 保存航线
7. 打开航线详情，展示地图回显和关联任务

**讲解**：
- "航线规划基于高德地图 API 实现可视化绘制"
- "支持地点搜索定位，鼠标点击连线形成巡防航线"
- "航线坐标以 JSON 格式自动存储"
- "详情页可回显航线走向，拖拽点位可微调"
- "已绑定未完成任务的航线受保护，无法删除"

---

## 五、任务管理 — 创建与启动（2分钟）

**操作**：
1. 进入任务管理页
2. 点击"新增"，填写任务名称
3. 下拉选择设备（只显示正常状态的设备）
4. 下拉选择航线
5. 填写执行人和开始时间
6. 保存任务（状态为"待执行"）
7. 点击"开始"，确认后任务状态变为"执行中"

**讲解**：
- "任务创建时绑定设备和航线，系统校验设备状态和航线占用"
- "任务状态包括待执行、执行中、已完成、已取消四种"
- "开始任务后设备状态自动切换为'任务中'"
- "任务详情可查看完整的状态变更时间线"

---

## 六、任务监控 — 模拟飞行（1.5分钟）

**操作**：
1. 点击执行中任务的"监控"按钮
2. 展示监控弹窗：地图 + 无人机图标移动 + 实时进度
3. 展示进度条增长和剩余时间变化
4. 点击"暂停飞行"，无人机停止，进度保持
5. 点击"继续飞行"，无人机继续移动
6. 等待或快进到任务完成

**讲解**：
- "监控界面使用高德地图 moveAlong 动画模拟无人机沿航线飞行"
- "无人机图标使用自定义 SVG，实时旋转朝向飞行方向"
- "进度条和预计剩余时间实时更新"
- "支持暂停和继续，暂停时保存当前进度到后端"
- "后端定时调度器每分钟自动递增执行中任务的进度"

---

## 七、AI 智能识别（2分钟）⭐核心亮点

**操作**：
1. 进入结果管理页，点击"新增"
2. 在图片上传区拖入或选择一张巡防图片
3. 等待 3-5 秒，AI 分析完成
4. 展示标注后的图片（目标框选 + 标签）
5. 展示自动填充的巡防概述、发现情况、处理情况
6. 补充或修改文字内容后保存
7. 打开结果详情，展示标注图片 + AI 识别文字

**讲解**：
- "这是系统的核心亮点——基于阿里千问 VL 大模型的 AI 智能识别"
- "上传巡防图片后，系统自动调用 Qwen2.5-VL-Max 进行图像分析"
- "AI 会检测图像中的目标物体并返回精确的边界框坐标"
- "后端使用 Java AWT 在图片像素上直接绘制标注框，精度高"
- "同时自动生成巡防概述、发现情况和处理建议，填入表单"
- "用户可以对 AI 结果进行人工复核和修改"

---

## 八、结果管理与报告导出（1分钟）

**操作**：
1. 返回结果列表，展示所有巡防结果
2. 点击某条结果的"详情"，展示完整信息
3. 点击"航线"按钮，地图回显本次巡防的飞行轨迹
4. 点击"PDF"按钮，下载巡防报告
5. 勾选多条结果，点击"导出"下载 Excel

**讲解**：
- "任务完成后自动生成结果记录，包含任务、设备、航线等快照信息"
- "结果详情可查看完整巡防概述、发现情况、处理情况"
- "地图回显本次巡防使用的航线轨迹"
- "支持单条 PDF 报告导出和批量 Excel 导出"
- "PDF 报告包含结果编号、任务信息、概述、发现、处理等完整字段"

---

## 九、权限管理（30秒）

**操作**：
1. 切换到普通用户账号登录
2. 展示普通用户只能查看，没有新增/编辑/删除按钮

**讲解**：
- "系统采用 RBAC 权限模型，管理员拥有全部操作权限"
- "普通用户仅拥有各模块的查看权限"
- "前端按钮和后端接口双重权限校验"

---

## 十、总结（30秒）

**操作**：回到首页

**讲解**：
- "本系统实现了无人机巡防管理的完整业务闭环"
- "从设备管理、航线规划、任务模拟到结果归档，全流程信息化"
- "AI 大模型的接入让巡防图像分析从人工升级为智能"
- "系统代码规范、架构清晰，适合教学演示和二次开发"
- "感谢各位评委老师，欢迎提问"
